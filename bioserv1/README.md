# Bioserv1 Module

This directory contains the Bioserver1 game server code used by this project.
It provides the core multiplayer game server logic and database-backed server state handling.

## Supported ENV variables

- `SERVER_IP` - Public or local IP address used by the game server.
- `DB_USER` - Database user.
- `DB_PASSWORD` - Database user password.
- `DB_DATABASE` - Database name.
- `DB_HOST` - Database host.
- `JAVA_DB_PARAMS` - Additional JDBC connection parameters.
